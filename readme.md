# Wall Of Mono

Interactively control a wall of 4 x 8 Monos.

## Endpoints

The web service has the following admistrative endpoints:

| Endpoint  | Function |
| --------- | -------- |
| `/status` | Returns the service status as JSON |
| `/pid`    | Returns the service's process id   |
